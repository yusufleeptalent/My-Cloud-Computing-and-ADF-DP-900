## 🔍 Overview  
This repository contains my Week 5 Cloud Computing & Azure Data Fundamentals work completed as part of the Data Technician programme.  
The project covers cloud concepts, cloud service models, Azure architecture, pricing, storage options, SQL databases, non‑relational data, real‑time analytics, and Microsoft Fabric.

---

## 🚀 Skills Demonstrated  

### ✔️ Cloud Computing Fundamentals  
- Understanding IaaS, PaaS, and SaaS  
- Benefits and risks of cloud adoption  
- Public, private, hybrid, and community cloud models  
- Real‑world use cases for organisations of different sizes  
- Cloud alternatives (on‑premises infrastructure)

### ✔️ Microsoft Azure Fundamentals  
- Azure regions, availability zones, resource groups  
- Azure Pricing Calculator cost estimation  
- Azure identity, governance, and security basics  
- Cloud cost management concepts  

### ✔️ Azure Data Services  

| Service | Description |
|--------|------------|
| 🟦 **Azure SQL Database** | Relational database for structured data workloads |
| 🌐 **Azure Cosmos DB** | Globally distributed NoSQL database |
| 💾 **Azure Storage Accounts** | Blobs, files, queues, and table storage |
| ⚡ **Azure Synapse & Fabric** | Large-scale analytics and real-time processing |
| 🔄 **Azure Data Factory** | Data pipelines and automation |
