# ☁️ Cloud Computing & Azure Project – Fundamentals, Architecture & Hands‑On Labs
*A project demonstrating core cloud computing concepts, Azure fundamentals, data services, and real‑world cloud solution design.*

## 🔍 Overview  
This repository contains my Week 5 Cloud Computing & Azure Data Fundamentals work completed as part of the Data Technician programme.  
The project covers cloud concepts, cloud service models, Azure architecture, pricing, storage options, SQL databases, non‑relational data, real‑time analytics, and Microsoft Fabric.

---

## 🚀 Skills Demonstrated  

### ✔️ Cloud Computing Fundamentals  
- Understanding IaaS, PaaS, and SaaS  
- Benefits and risks of cloud adoption  
- Public, private, hybrid, and community cloud models  
- Real‑world use cases for organisations of different sizes  
- Cloud alternatives (on‑premises infrastructure)

### ✔️ Microsoft Azure Fundamentals  
- Azure regions, availability zones, resource groups  
- Azure Pricing Calculator cost estimation  
- Azure identity, governance, and security basics  
- Cloud cost management concepts  

### ✔️ Azure Data Services  
- **Azure SQL Database** – relational data workloads  
- **Azure Storage Accounts** – blobs, tables, queues, files  
- **Azure Cosmos DB** – NoSQL, globally distributed databases  
- **Azure Synapse & Fabric** – large‑scale analytics, real‑time processing  
- **Azure Data Factory** – data pipelines & automation  

### ✔️ Power BI & Fabric Integration  
- Exploring data analytics in MS Fabric  
- Working with real‑time analytics environments  
- Understanding how Power BI connects to Azure resources  

### ✔️ Hands‑On Cloud Labs  
Completed practical labs including:

- Getting data into Azure services  
- Working with Azure SQL via Skillable labs  
- Exploring storage accounts & Cosmos DB  
- Real‑time analytics, data pipelines, and Fabric experiences  
- DP‑900 Azure Data Fundamentals practice exam
